Index of FluidMechanics/Spectral-CompactDiff-DNS/simple/

Notice: - compile with cpl make scddns - run as scddns [] - defaults to scddns0.dat if omitted - the input file contains a number of parameters of the form name=value, optionally followed by a velocity field as binary data - if no binary data are present, a random initial field is generated - output velocity fields, sequentially numbered as scddns.dat, are in the same format as the input file; any of them can be used to restart the computation.
