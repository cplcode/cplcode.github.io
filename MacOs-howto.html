CPL-on-Macintosh howto

Step by step instructions to use CPL on Apple MacOs.
1 - Get Command Line Tools.
To compile anything (not just CPL) on Macintosh you need the Apple Command Line Tools, a subset of the Xcode programming anvironment. You may already have them if for other reasons you installed Xcode. Otherwise you can use the following instructions, modelled after https://osxdaily.com/2014/02/12/install-command-line-tools-mac-os-x/. Open a terminal: if you do not already have the terminal app in your taskbar, write terminal in the search box and click terminal.app.
2 - Install Command Line Tools:
Enter 
xcode-select --install
A dialog box will appear asking if you want to install the Command Line Developer Tools. Accept. If the box does not appear, you probably have them already. You do not need the (much larger) Xcode package unless you want it for other purposes.
3 - Install the Homebrew package manager:
Follow instructions on the brew.sh website.
4 - Install original gcc (the one provided by Apple is an alias for their own clang):
brew install gcc
5 - Unpack the CPL archive you previously downloaded, with the CC environment variable pointing at your new C compiler:
CC=gcc-10 sh Downloads/*cpl.sh
6 - For graphics, install gnuplot:
brew install gnuplot

You are all set!
