CPL-on-Debian-on-Windows howto

Step by step instructions to using CPL with Microsoft Windows 10.
1 - Enable WSL:
To enable the Windows Subsystem for Linux (WSL) of Microsoft Windows 10, you can use a sequence of Powershell terminal commands (described here), or alternately open the Control Panel, then Programs and Features, then Turn Windows Features On or Off, and check the case for Windows Subsystem for Linux.
2 - Install Debian on WSL:
Open Microsoft Store, enter debian in search box, install the free Debian app (or your favourite other Linux distribution); launch the app.
3 - Bring Debian up to date. In the Debian terminal type:
sudo apt update
sudo apt upgrade
4 - Install needed packages gcc, make, info, curl:
sudo apt install gcc make info curl
5 - Copy the CPL archive you previously downloaded from your browser to your new Debian home:
cp /mnt/c/Users/yourusername/Downloads/*cpl.sh ./
(where yourusername must be replaced by your own Windows user directory, enclosed in quotes if it contains spaces).
6 - Unpack the cpl archive:
sh *cpl.sh
You may need to close and re-open the Debian window before the newly installed commands become available at the command prompt. You can open more than one Debian window if you want, for instance, to edit your own CPL program in one and to execute it in the other.

You are all set!

Notice about graphics: Microsoft's Windows Subsystem for Linux is terminal-only (no window manager), and gives a limited experience of a Linux distribution. The CPL compiler, interpreter and editor are all built to run in a terminal and will fit in seamlessly, but if the program you write draws graphics and uses the additional gnuplot package, installing it through Debian (apt install gnuplot) will not do. Instead you must install the Windows version of gnuplot, available from the official gnuplot site, and then

7 - link gnuplot to your Debian setup. After you have downloaded and installed gnuplot for Windows, open a Debian terminal and type (with attention to spaces):
ln -s '/mnt/c/Program Files/gnuplot/bin/gnuplot.exe' bin/gnuplot
Now you can open gnuplot as just gnuplot in your Debian terminal, and CPL graphics will automatically find it too.
