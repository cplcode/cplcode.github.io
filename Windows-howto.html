CPL-on-Debian-on-Windows howto

1 - Enable WSL
To enable the Windows Subsystem for Linux (WSL), available in Microsoft Windows 10, you can use a sequence of Powershell terminal commands (described here), or more simply open the Control Panel, then Programs and Features, then Turn Windows Features On or Off, and check the case for Windows Subsystem for Linux.
2 - Install Debian
Open Microsoft Store, enter debian in search box, install the free Debian app (or your favourite other Linux distribution); launch the app.
3 - Upgrade Debian to current updates
In the Debian terminal type:
sudo apt update
sudo apt upgrade
4 - Install gcc, make and info
sudo apt install gcc make info
5 - Copy the downloaded CPL archive to your Linux home
cp /mnt/c/Users/yourusername/Downloads/*cpl.sh ./
6 - Unpack the CPL archive
sh *cpl.sh

You are all set!

Note: Microsoft's Windows Subsystem for Linux is terminal-based and excludes graphics. Graphics can be provided through a non-official X server, but if the application you foresee is graphical, you may prefer to start from scratch with a non-Microsoft Linux distribution. On the other hand, WSL offers a quick start off of CPL on Windows, and if you find it useful you can always upgrade to a complete Linux later.
