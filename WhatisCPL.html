What is CPL?

Three-letter acronyms collect many meanings. CPL has dozens, for example several cities' Public Libraries, a few Cricket Premier Leagues, a Commercial Pilot License, a Microsoft Windows Control PaneL file, and an ancient programming language that was the ancestor of BCPL and C. Here we add five more:

  • Concealed Pointer Lookup
  • Custom Parameter Lists
  • Consistent Procedure Linkage
  • Compiled Programming Language
  • Conceived by Paolo Luchini

The present CPL, unrelated to the Cambridge or Combined Programming Language, is a general-purpose programming language akin to FORTRAN or C (but in fact even more similar to the once common Pascal and Modula-2, for those who remember them), with some unique features. (**p).col=j;

It also aims for the ease of use of interpreted languages like MATLAB or Python, without giving up the error-catching benefits of strong typing and the sheer speed of compiled execution. CPL was thought out as a compiled language from the very beginning; when used for computationally intensive tasks, it has the speed of compiled C. >makecpl Helloworld

In fact, its compiler is implemented as a nimble translator to C which then invokes the system C compiler; this action, as well as the re-making of any required modules that were modified in the meanwhile, occurs behind the curtains at a single makecpl command. Only much later the icpl interpreter was developed, with the ability to run a combination of interpreted and compiled CPL modules, as well as any function provided by an existing C library, interactively from its console.

To cap this introduction, a hallmark aesthetic touch of CPL p(i)=EXP(-{[x(i)-x(0)]/sigma}^2/2) is the availability of all three kinds of brackets for grouping arithmetic (or other) expressions, just as you were taught in school. While this might be an unessential feature to some, alternating brackets so much eases the human eye reading and proofreading nested parentheses.
