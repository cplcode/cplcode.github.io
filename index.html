A Compiled Programming Language

makecpl

Write your program using your favorite text editor, compile it with makecpl <yourprogram>, execute <yourprogram>.

icpl

Interactive cpl: evaluate any CPL expression, either on the command line as in icpl <expression>, or in the icpl interpreter.

edit

When invoked under the edit alias, icpl turns into a terminal-screen text editor providing CPL word and syntax completion.
